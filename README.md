# 🧀🔺 UAI_Buntu v0.4.3

Uma distribuição Linux customizada baseada no Ubuntu, desenvolvida por um programador de 11 anos usando o Cubic. O sistema traz um visual escuro moderno com detalhes elegantes em amarelo e marrom, Fastfetch personalizado nativo no terminal e é totalmente limpo de logotipos antigos!

---

## 📥 Como Baixar e Instalar / How to Download and Install

Como o arquivo final é grande, o sistema foi dividido em **5 partes (.zip)** na nossa área de [Releases](https://github.com). Baixe todas as partes para a mesma pasta e siga as instruções abaixo:

### 🐧 No Linux (Terminal)
Baixe as 5 partes na sua pasta de `Downloads` e rode este comando único para juntar e extrair a sua ISO original:
```bash
cd ~/Downloads && cat UAI_Buntu_part*.zip > UAI_Buntu_completo.zip && unzip UAI_Buntu_completo.zip
```

### 🪟 No Windows (Interface Gráfica)
1. Baixe as 5 partes na mesma pasta (ex: `Downloads`).
2. Clique com o **botão direito do mouse apenas no primeiro arquivo** (`UAI_Buntu_part00.zip`).
3. Selecione o **7-Zip** ou **WinRAR** e clique em **"Extrair Aqui"** (*Extract Here*).
